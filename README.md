📘 Student Marks Management System

A full-stack web application designed to simplify student marks management with two separate dashboards – one for Teachers and one for Admins.

🚀 Features
👩‍🏫 Teacher Dashboard

Upload and manage student marks for their assigned subjects.

View the list of subjects they teach.

Generate individual student reports.

🧑‍💼 Admin Dashboard

Add and manage teachers.

Add, update, and manage students.

Create and manage courses and subjects.

Export final consolidated marksheets uploaded by teachers.

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Spring Boot (Java)

Database: MongoDB

📂 Project Structure

Admin Module → Handles teacher, student, course, subject, and marksheet management.

Teacher Module → Allows teachers to upload marks and generate reports.

🎯 Purpose

This project was developed as part of a college academic project to practice full-stack development and implement a role-based system for effective marks management.
